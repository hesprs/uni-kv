This is the TypeScript monorepo for Unified KV, `blueprint/` rests the canonical spec this project wants to achieve.

This project aims to deliver a lightweight and unified interface for data storage.

## Commands

- `bun lint`: format and fix fixable lint errors (always run before `bun check`).
- `bun check`: check types, lint and format (no file change).
- `bun test`: run tests.
- `bun run build`: build.

## Code Quality

- No non-null assertion
- No explicit `any`
